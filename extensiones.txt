modificación para los mapping.

Extensiones para consultar o para como remplazarla a nuestra configuración:

Extensión (nvchad.tubfline) - movimiento de buffer ficheros adelante y atrás

modificar extension de nvchad.tunfline por bufferline.

las líneas:

vim.keymap.set("n", "<Tab>", ":bnext<CR>", { desc = "Next buffer" })
vim.keymap.set("n", "<S-Tab>", ":bprev<CR>", { desc = "Previous buffer" })
vim.keymap.set("n", "<leader>x", ":bd<CR>", { desc = "Close buffer" })

Extensión (nvchad-term):

para que se utiliza: es utilizada para poder abrir termianles dentro neovim con la configuracion propia de nvchad.

	para poder adaptarlo a nuestra configuración lo podemos poner con estas líneas

	vim.keymap.set("n", "<leader>tv", function()
	  vim.cmd("vsplit | terminal")
	end, { desc = "Terminal vertical" })

	vim.keymap.set("n", "<leader>th", function()
	  vim.cmd("split | terminal")
	end, { desc = "Terminal horizontal" })
